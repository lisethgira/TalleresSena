Ejemplos de Etiquetas

Etiqueta <mark>:

La etiqueta <mark> se utiliza para resaltar un fragmento de texto en el contenido. Aquí hay un ejemplo:

<p>Este es un párrafo de ejemplo<mark>resaltado</mark></p>

Este es un párrafo de ejemplo resaltado.


Etiqueta <meter>:

La etiqueta <meter se utiliza para mostrar una medida en una escala. Aquí hay un ejemplo:

<p>Calificación:<meter value="8" min="0" max="10">8 de 10</meter></p>

Calificación: 8 de 10

En este ejemplo, el atributo "value" especifica el valor de la medida, mientras que los atributos "min" y "max" definen los límites inferior y superior de la escala.


Etiqueta <dialog>:

La etiqueta <dialog> se utiliza para crear una ventana emergente o un diálogo en la página. Aquí hay un ejemplo:

<dialog open><h3>¿Estás seguro?</h3><p>¿Quieres eliminar este archivo?</p><button>Cancelar</button><button>Eliminar</button></dialog>

¿Estás seguro?

¿Quieres eliminar este archivo?

En este ejemplo, el contenido del diálogo se incluye entre las etiquetas <dialog> y </dialog>. Los botones proporcionan opciones para el usuario.


Etiqueta <time>:

La etiqueta <time> se utiliza para mostrar una fecha y hora. Aquí hay un ejemplo:

<p>Publicado: <time datetime="2022-05-23T12:34:56Z">23 de mayo de 2022</time></p>

Publicado: 23 de mayo de 2022

En este ejemplo, el atributo "datetime" se utiliza para especificar la fecha y hora en formato ISO 8601. El contenido visible del elemento es el texto "23 de mayo de 2022".


Etiqueta <figure> y Etiqueta <figcaption> :

La etiqueta <figure> se utiliza para agrupar contenido multimedia y proporcionar una leyenda.

La etiqueta <figcaption> se utiliza para proporcionar una leyenda o descripción para el contenido multimedia agrupado por la etiqueta <figure>.

Aquí hay un ejemplo:

<figure><img src="src/image/imagen.jpg" alt="Cohete Rojo" style="border: 1px solid black; width: 200px;"><figcaption>Figura 1: Cohete rojo.</figcaption></figure>
