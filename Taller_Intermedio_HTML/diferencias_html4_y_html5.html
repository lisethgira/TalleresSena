Diferencias entre HTML 4 y HTML 5:

  1. Doctype:

    En HTML 4, el Doctype se define como <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">, mientras que en HTML 5, se define como <!DOCTYPE html>.

  2. Elementos semánticos:

    HTML 5 incluye nuevos elementos semánticos como header, footer, nav, article, section, aside, etc. Estos elementos ayudan a los desarrolladores a crear una estructura de página web más clara y significativa.

  3. Formularios:

    En HTML 5, se agregaron nuevos tipos de entrada de formulario, como email, url, date, number, range, etc.

  4. Detección automática de codificación de caracteres:

    HTML 5 puede detectar automáticamente la codificación de caracteres del documento, lo que significa que los desarrolladores ya no necesitan especificar la codificación de caracteres en la etiqueta meta.

  5. Vídeo y audio:

    En HTML 5, se agregaron nuevos elementos para la inclusión de contenido multimedia como video y audio, lo que permite a los desarrolladores integrar fácilmente contenido multimedia en sus páginas web sin necesidad de plug-ins de terceros.

  6. Atributos alt y title:

    En HTML 5, se recomienda usar el atributo title para proporcionar información adicional sobre un elemento, mientras que el atributo alt se usa exclusivamente para describir una imagen o un objeto en caso de que no se muestre correctamente.

En resumen, HTML 5 ofrece una serie de mejoras significativas en términos de semántica, funcionalidad y facilidad de uso en comparación con HTML 4.
